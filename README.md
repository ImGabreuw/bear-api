## 🚀 Installation

```bash
# Repository clone
$ git clone https://github.com/ImGabreuw/beer-api

# Go to the application directory
$ cd beer-api/

# Start the application
$ mvn spring-boot:run
```

Go to http://localhost:8080/api/v1/beer
